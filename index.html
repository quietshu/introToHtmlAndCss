为什么要学习 HTML？

你看到的每一个网页都是用一种叫做 HTML 的语言编写的，HTML 定义了网页的结构。

网页信息在传入电脑时，只是一串 HTML 代码。浏览器的作用便是将 HTML 显示成我们看到的界面。

HTML 是网络技术的基础之一。

HTML 基本语法 
	        
<!doctype html>	
声明文档类型为 HTML

<html>						
尖括号相当于 begin 和 end

	...
	<body>					
页面本体

		...
	</body>	
</html>
一个简单的例子 
	        
<!doctype html>	
<html>	
	<body>	
    <p>hello, world</p>									
    <p><i>hello, world</i></p>				
    <p><strong>hello, world</strong></p>	
      
p：一段文字，i：斜体，strong：加粗

	</body>	
</html>
demo

HTML 具有许多用途不同的标签

标签具有属性，例如 <p align="center">...</p>

标签可以互相嵌套，标签的特性会传递给其儿子

大部分标签成双出现，开、闭标签需要按照结构对应

总的来说，按嵌套盒模型（HTML 的层次结构），HTML 清晰地表示了一个网页的逻辑

Web 编程很注重分工：


逻辑与样式是分开的

样式就得靠 CSS 了 - CSS（层叠样式表）
CSS 初步

单靠 HTML 无法建出美观的网页

CSS 的作用是描述 HTML 元素的外观

<p style="color:red">hello</p> → hello

<p style="font-size:50px">50像素</p> → 五十像素

<p style="text-decoration:underline">下划线</p> → 下划线


位置、颜色、字体、大小、间距、…… → CSS

CSS 与 HTML

HTML 在 body 标签开始前，有一个 head 标签，用来声明此 HTML 的关联文件等信息

<link href="style.css" rel="stylesheet" />

调用与此 .html 文件同一目录下的 style.css

style.css 
		
p {
  color: red;
  font-style: italic;
}
p.myClass {
  color: blue;
}
demo

引用了 style.css 的 .html 文件显示效果

<p>hello</p> → hello

<p class="myClass">hello</p> → hello

class 和 id 两个常用属性相当于标签的标示符。也可以理解为，标签名是公共类型，id 是变量名，class 是自定义的类型
